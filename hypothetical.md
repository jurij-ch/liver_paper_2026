Good Prognosis: 
 - Isolated MASLD - There are no laboratory changes, visible only on ultrasound. Elastography - F0

Moderate Prognosis:
 - MASLD-Cardio - (Lipid metabolism) - Changes in Liver markers and / or ipid markers, Coronary heart disease positive anamnesis, MASLD visible on ultrasound. Elastography - F0 
 - MASLD-Diabetes - (Glucose tolerance disorder) - Changes in Liver markers and / or glucose markers, MASLD visible on ultrasound. Elastography - F0
 - MASLD-Thyroid - (Thyroid gland) - Changes in Liver markers and / or thyroid markers, MASLD visible on ultrasound. Elastography - F0 
 - MASLD-Nephrologic - (Kidney) - Changes in Liver markers and / or MASLD visible on ultrasound. Elastography - F0

Poor Prognosis (1):
 - Inflammatory MASLD - (Inflammation) - Changes in Liver markers, inflammation markers.  Elastography - F0 or greater
Poor Prognosis (2):
 - MASLD-Mixed - Changes in multiple profile markers, MASLD visible on ultrasound. Elastography - F0
